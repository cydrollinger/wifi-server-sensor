The ANT+ network key is 8 hex values and is shown below:

0xB9, 0xA5, 0x21, 0xFB, 0xBD, 0x72, 0xC3, 0x45


The ANT-FS key is 8 hex values and is shown below:

0xA8, 0xA4, 0x23, 0xB9, 0xF5, 0x5E, 0x63, 0xC1

These keys are used in the ANT_SetNetworkKey() function as described in section 9.5.2.7 of the
ANT Message Protocol and Usage document.  You will find the latest version of this document,
and other useful information in the Developer's Zone on the ANT website at www.thisisant.com.
